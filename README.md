# KeaBuilder AI Assessment

## 📌 Overview
This project demonstrates a simple AI system for KeaBuilder, including:
- Lead classification (Hot/Warm/Cold)
- AI-based response generation
- Content routing (image/video/voice)
- Similarity matching
- Basic fallback handling

---

## ⚙️ Features

### 1. Lead Processing
- Classifies leads based on input fields
- Generates personalized responses

### 2. Content Routing
- Routes requests to different providers:
  - Image
  - Video
  - Voice

### 3. Similarity Matching
- Uses TF-IDF + cosine similarity
- Finds closest matching user input
